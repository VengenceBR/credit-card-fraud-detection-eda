# 💳 Credit Card Fraud Detection - Exploratory Data Analysis

![Banner](banner.png)

## 📌 Project Overview

This project presents an **Exploratory Data Analysis (EDA)** of the Credit Card Fraud Detection dataset from Kaggle. The analysis explores transaction patterns, fraud distribution, transaction amounts, and feature relationships using Python.

---

## 📂 Dataset

**Source:** Kaggle

**Dataset:** Credit Card Fraud Detection Dataset

The dataset contains credit card transactions made by European cardholders in September 2013.

- **Total Transactions:** 284,807
- **Fraudulent Transactions:** 492
- **Features:** 31
- **Target Variable:** `Class`

---

## 🎯 Objectives

- Explore the dataset structure
- Identify missing values
- Analyze fraud distribution
- Examine transaction amount statistics
- Visualize transaction patterns
- Study correlations between numerical features

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Visualizations

The project includes:

- Fraud vs Legitimate Transactions (Bar Chart)
- Fraud Distribution (Pie Chart)
- Transaction Amount Histogram
- Correlation Heatmap
- Transaction Amount Boxplot
- Transaction Amount by Class
- Transaction Time Distribution

---

## 📈 Key Insights

- The dataset is highly imbalanced.
- Fraudulent transactions account for less than 1% of all transactions.
- Most transactions involve relatively small amounts.
- Significant outliers exist in transaction amounts.
- Most numerical features have weak correlations with one another.
